D2D — Android Logistics Management

SmartKargo Pvt. Ltd. — Nov 2024 – Present

Overview

The D2D Logistics App is a logistics and delivery management mobile application for airline cargo operations. The project focused on modernizing the legacy UI, improving routing, and reducing dependency costs while increasing reliability.

What I built

  • Migrated the UI from legacy layouts to Material Design using Jetpack Compose to improve usability and consistency across devices.
  • Implemented a custom route generation system for drivers that eliminated dependence on Google Maps APIs — reducing route-related costs by ~90%.
  • Replaced an IoT-based driver tracking approach with an in-house location solution to cut operational costs by ~87%.
  • Integrated Azure Blob Storage for images and replaced bitmap-based uploads to achieve clearer images and more reliable uploads.

What this project taught me

This project deepened my understanding of Android UI modernization, performance optimization, and system design for cost efficiency. I learned to reason about trade-offs (accuracy vs cost for location tracking), implement secure API integrations, and handle large files using cloud storage.

Technologies

Java · Kotlin · Jetpack Compose · Retrofit · OkHttp · Azure Blob Storage · SQLite/Room · JWT/OAuth
