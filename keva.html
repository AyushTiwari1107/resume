Keva Fragrance — CRM Backend

Daten And Wissen Pvt. Ltd.

Overview

Keva Fragrance CRM involved building backend services to handle bulk data uploads, processing, and API-driven integrations supporting the CRM operations.

What I built

  • Developed a comprehensive backend to manage bulk uploads and data processing for the Keva Fragrance application.
  • Designed and implemented RESTful APIs to enable efficient and reliable data transactions between frontend and backend systems.
  • Improved database schema and query performance, reducing average response times by approximately 20%.
  • Integrated CI/CD with Jenkins and used SonarQube for code quality monitoring during deployment pipelines.

What this project taught me

Working on Keva taught me backend design at scale, efficient schema design, and the importance of CI/CD and code quality tooling to maintain production-grade systems.

Technologies

REST · MySQL · Jenkins · SonarQube · Java · CI/CD
